Project: Architectur Site

This web application was built to make calculations in the work of architect, and provides some information about building law.

The project is an ASP.NET Core 5 web application based on MVC structure. The site uses JQuery and Bootstrap. Tests were done using Xunit. The site is in polish and has only polish building law included in the calculations.

Application features:

  • Stairs calculation
  • Customizing results
  • Table with polish stairs building law
